The Code for FIZZ BUZZ.


                

                

                    
The Code is structured in three functions.

                    
GetValues

                    
getValues() accepts the user input from the page. It utilizes getElementById to pull the values from the page. Then it validates the input to make sure is valid and passes those values to the fizzBuzz() function. The function FizzBuzz returns an array of values and passes that array to the displayData() function.

                    
FizzBuzz

                    
The fizzBuzz() function is used to check the condition for Fizz, Buzz, or FizzBuzz. We use the modulo operator (%). to check if a number is divisible by the Fizz or/and Buzz values. This gives us four possible combinations.

                    
    
                        
  • The number is divisible by Fizz and Buzz, which makes it FizzBuzz.
    • 
                        
  • The number is divisible by Fizz.
    • 
                        
  • The number is divisible by Buzz.
    • 
                        
  • The number is not divisible by neither Fizz or Buzz.
    • 
                    

                    
This function returns an array with all the values.

                    
DisplayData

                    
displayData() is used to inject the data into the page. It takes an array with the FizzBuzz data as parameter. This data is then divided in groups of five elements to form the rows that will be injected to the table. A template was used to facilitate the reuse and construction of the table rows.
